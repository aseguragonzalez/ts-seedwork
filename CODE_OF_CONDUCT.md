# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) (v2.1) as its code of conduct.

All contributors are expected to uphold these standards. Instances of unacceptable behavior may be reported to [a.segura.gonzalez@gmail.com](mailto:a.segura.gonzalez@gmail.com).

The project maintainer is responsible for clarifying and enforcing the standards and may take any action deemed appropriate in response to reported behavior.
